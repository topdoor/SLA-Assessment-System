🔗 고객클레임 점수 자동 반영
클레임 DB에서 거점별 배송지연·오배송·오출하·불량파손 건수를 자동 집계하여
항목별 발생율(%) 계산 후 SLA 점수를 반영합니다.
점수 기준   0.1% 이하 → 10점  |  0.1%~0.2% → 8점  |  0.2%~0.3% → 6점  |  0.3%~0.5% → 4점  |  0.5% 이상 → 2점

🔧 Apps Script 배포 가이드
1. Google Sheets 열기 → 확장 프로그램 → Apps Script 2. SLA_AppsScript.gs 내용 전체 붙여넣기 → 저장 (Ctrl+S) 3. 함수 선택: setupAll → 실행 → 시트 8개 + 초기 데이터 자동 생성 4. 배포 → 새 배포 → 유형: 웹앱 → 다음 사용자로 실행: 나 → 액세스: 모든 사용자 (익명 포함) → 배포 → URL 복사 5. 위 "연동 URL 설정"에 URL 붙여넣기 → 저장
